# Kairos Deprecation

Kairos is the old system powering the GoTrain instance hosted at https://kairos.knijn.one/gotrain. This is being deprecated due to age, and in order to provide a better experience. It was based on an old project for school that I am no longer maintaining. The old instance will stay up as long as possible, but no realistic effort will be made to keep it up if it goes down.

## Replacement
I will be replacing the old instance with a different instance of GoTrain, on another (new) server. This is available at https://gotrain.knijn.one. I suggest migrating to it, if you were relying on the old GoTrain instance. The [status page](https://status.knijn.one) has been updated appropriately, with the old instance indicated as *legacy*.

If you have any concerns regarding this change, please reach out to me.
